# Portugal Trip 2026

Trip planner for **July–August 2026**: Toronto → Porto → Mom's (Coimbra) → Lisbon & Madeira → home.

## Live site (share with Lisete)

**After enabling GitHub Pages:**  
**https://mexemexe02.github.io/portugal-trip-2026/**

### How to make it live (one-time)

1. Open the repo: **https://github.com/mexemexe02/portugal-trip-2026**
2. Go to **Settings** → **Pages** (under “Code and automation”).
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose **Branch:** `master`, **Folder:** `/ (root)`. Click **Save**.
5. In 1–2 minutes the site will be available at the link above.

## What’s on the site

- **Home** – Hero, dates, cost range, **book tickets** links (Google Flights, Expedia, KAYAK, TAP), and links to all pages.
- **Itinerary** – Flow, dates, interactive checklist, accommodation.
- **Flights & pricing** – Cost snapshot, price charts, confirmed prices, and all booking links.
- **Full plan** – All **7 options (A–G)** with flow, bookings, pros/cons.
- **Compare options** – Interactive comparison of all options so you can choose one.
